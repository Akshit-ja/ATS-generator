## Project Overview
The application is an AI-powered resume generator with the following stack:
- **Frontend**: Next.js with TypeScript and Tailwind CSS.
- **Backend**: Python with FastAPI.
- **Database**: PostgreSQL (managed via Supabase).
- **Asynchronous Tasks**: Celery with a Redis message broker.
- **AI/NLP**: OpenAI API (GPT-4), spaCy, and Sentence-Transformers.
- **Containerization**: Docker and Docker Compose for the local environment.

## Your Mission
